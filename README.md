# 🏏 IPL Complete Analysis Dashboard

An interactive data analysis dashboard built with Python and Streamlit.

## 📊 What this project does
- Analyzes 15+ years of IPL data (2008-2024)
- Top 10 winning teams across all seasons
- Top run scorers & wicket takers
- Player of the Match analysis
- Toss impact on match results

## 🛠️ Tools Used
Python | Pandas | Matplotlib | Seaborn | Streamlit
